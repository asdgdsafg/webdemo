正则表达式(regular expression)

1. 创建正则表达式

var box = new RegExp('box');第一个参数字符串
var box = /box/;同上，为简写方式
var box = new RegExp('box', 'i');第二个参数可选模式修饰符
var box = /box/i;同上，为简写方式

2. 模式修饰符的可选参数

var box = /box/i;i 忽略大小写
var box = /box/g;g 全局匹配
var box = /box/m;m 多行匹配
var box = /box/igm;igm 全部使用
lastIndex该属性存放一个整数，它声明的是上一次匹配文本之后的第一个字符的位置；W3C详细解释

3. RegExp对象的方法

test()/[a-z0-9]/.test(str) 在字符串中测试模式匹配，返回 true 或 false
exec()返回一个数组，其中存放匹配的结果。如果未找到匹配，则返回值为 null；W3C详细解释

4. String 对象中的正则表达式方法

match(pattern)返回 pattern 中的子串或 null
replace(pattern, replacement)用 replacement 替换 pattern
search(pattern)返回字符串中 pattern 开始位置
split(pattern)返回字符串按指定 pattern 拆分的数组

5. 元字符

5.1 单个字符
.匹配除 "\n" 之外的任何单个字符；要匹配包括 '\n' 在内的任何字符，请使用像"(.|\n)"的模式
[abc]匹配所包含的任意一个字符。例如， '[abc]' 可以匹配 "plain" 中的 'a'
[^abc]匹配未包含的任意字符。例如， '[^abc]' 可以匹配 "plain" 中的'p'、'l'、'i'、'n'
[a-z0-9]匹配括号中的字符集中的任意字符
[^a-z0-9]匹配非括号中的字符集中的任意字符
\d匹配数字同 [0-9]
\D匹配非数字同 [^0-9]
\w匹配字母和数字及_同 [a-z0-9_]
\W匹配非字母和数字及_同 [^a-z0-9_]
5.2 空白字符
\b匹配空格字符
\f匹配一个换页符
\n匹配换行符
\r匹配一个回车符
\t匹配制表符
\s匹配空白字符、空格、制表符和换行符；等价于 [ \f\n\r\t\v]
\S匹配非空白字符
5.3 锚字符
^行首匹配
$行尾匹配
\A只有匹配字符串开始处
\b匹配单词边界，也就是指单词和空格间的位置；例如， 'er\b' 可以匹配"never" 中的 'er'，但不能匹配 "verb" 中的 'er'；词在[]内时无效
\B匹配非单词边界，'er\B' 能匹配 "verb" 中的 'er'，但不能匹配 "never" 中的 'er'
\z只匹配字符串结束处
abc|def|ghi匹配 abc 或 def 或 ghi 中任意一个；当匹配的单词都需要行首、行尾匹配时，需要括号括起来/^(abc|def|ghi)$/
5.4 重复字符
n?匹配 0 个或 1 个 n
n*匹配 0 个或任意多个 n
n+匹配至少一个 n
(abc)?匹配 0 个或 1 个(abc)
n{x}匹配 x 个 n
n{x,}匹配至少 x 个 n
n{x, y}匹配最少 x 个、最多 y 个 n
5.5 分组字符
(abc)匹配 abc
\1 | $1必须配合(abc)\1匹配第 1 个括号里的内容；例/^(.)\1+$/匹配一串同个字的字符串，如aaaa、啊啊啊啊等等
\2 | $2必须配合(abc)(deg)\2匹配第 2 个括号里的内容
\3 | $3必须配合(abc)(def)(ghi)\3匹配第 3 个括号里的内容

6. 常用正则

/^[0-9]{n, y}$/匹配数字，至少n个数字，最多y个数字
/^[a-z]{n, y}$/匹配字母，至少n个字母，最多y个字母
/^[a-z0-9]{n, y}$匹配字母或数字，至少n个字母或数字，最多y个字母或数字
/^\s$/匹配空白字符、空格、制表符和换行符
/^[a-z0-9][a-z0-9_.-_]{n, y}@[a-z0-9]{n, y}.[a-z]{2, 5}$/i邮箱
