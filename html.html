2. 有属性标签

a超链接
必选属性href指定超链接目标的
绝对URL - 指向另一个站点相对URL - 指向站点内的某个文件锚点指向页面中的锚
可选属性target规定在何处打开链接文档
_blank在新窗口打开
_parent在父框架集中打开
_self默认
_top在整个窗口中打开
framename在指定的框架中打开<frame>的name属性对应
download下载的超链接 href="fileURL" download="filename"
id(name)锚点定位 href="#id" id="id"
rel
abbr单词或短语的缩写形式
必选属性title有关元素的额外信息
audio音频
必选属性srcURL
可选属性autoplay音频在就绪后马上播放
controls向用户显示控件，比如播放按钮
loop每当音频结束时重新开始播放
muted视频输出应该被静音
preload音频在页面加载时进行加载，并预备播放
img图片
必选属性alt图片信息
src图片URL
可选属性height高度
width宽度
ismap将图像定义为服务器端图像映射
longdesc指向包含长的图像描述文档的 URL
usemap将图像定义为客户器端图像映射
meta定义页面的各种元数据
必选属性content定义与 http-equiv 或 name 属性相关的元信息
可选属性http-equiv把 content 属性关联到 HTTP 头部
scheme用于翻译 content 属性值的格式
name把 content 属性关联到一个名称
用法<meta charset="utf-8">
<meta name="author" content="作者">当前页面的作者
<meta name="description" content="HTML5 页面">当前页面的描述
<meta name="keywords" content="html5,css3">当前页面的关键字
<meta name="generator" content="txt">当前页面的编码工具
<meta http-equiv="refresh" content="5;http://">5 秒后跳转到指定 URL
<meta name="robots" content="html5,css3">搜索引擎抓取
<meta http-equiv="content-type" content="text/html charset=utf-8">另一种声明字符编码的方式
<meta name="viewport" content="width=device-width">响应式
bdo覆盖默认的文本方向
必选属性dir文字的方向从左到右（ltr），还是从右到左（rtl）；从左到右 | 从右到左

4. 全局属性

id元素的唯一的名称<p id="id">只有我叫 id</p>
class某个或多个元素名称<p class="class">我是其中一个叫 class 的</p>
title关于元素的额外信息<p title="提示">悬停 2 秒有提示信息</p>
hidden隐藏元素<p hidden>我被隐藏了</p>
stylecss 样式<p style="color: red">我的字体是红色的</p>
lang局部设置语言<p lang="en">我被更改了语言</p>
accesskey快捷键，windows 下 alt+指定键，需要浏览器不冲突<a href="https://www.bilibili.com/" accesskey="b">alt+b 跳转b站</a>
contenteditable让文本处于可编辑状态，false 则不可编辑<span contenteditable="true"></span>点击后，修改我
dir让文本的对齐方式靠左（ltr），还是靠右（rtl）<p dir="rtl"></p> 靠左靠右
tabindex表单中按下 tab 键，实现获取焦点的顺序如果是-1，则不会被选中 <input type="text" tabindex="1">
