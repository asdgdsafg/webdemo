2D、3D

所有变化前后的图形的中心点都不变，可使用transform-origin改变中心点

只有设置了perspective的值元素才会有Z轴，perspective: none元素则没有Z轴(就是2D元素)

1. 旋转

transform: rotate(180deg);rotate()括号里为度数，正值为顺时针方向旋转，负值相反，deg是degree的缩写，意为角度
transform: rotateX(180deg);rotateX()正值为从下到上翻转
transform: rotateY(180deg);rotateY()正值为从左到右翻转
transform: rotateZ(180deg);rotateZ()和rotate()一样
transform: rotate3d(x, y, z, deg);rotate3d()4个参数，xyz的值为-1、0或1，-1为反方向，0为不旋转，1为正方向旋转，deg为度数

2. 缩放

transform: scale(0.8, 0.8);scale()宽高都缩小为0.8倍
transform: scaleX(0.8);scaleX()宽缩小为0.8倍
transform: scaleY(0.8);scaleY()高缩小为0.8倍
transform: scaleZ(0.8);scaleZ()需要配合rotateX()使用
transform: scale3d(x, y, z);scale3d()需要配合rotateX()使用

3. 移动

transform: translate(100px, 100px);translate()从当前位置向XY轴各移动100px
transform: translateX(100px);translateX()从当前位置向X轴移动100px
transform: translateY(100px);translateY()从当前位置向Y轴移动100px
transform: translateZ(100px);translateZ()从当前位置向Z轴移动100px
transform: translate3d(x, y, z);translate3d()从当前位置向xyZ各轴移动

4. 斜拉

transform: skew(10deg, 10deg);skew()从当前位置向XY各斜拉10度
transform: skewX(10deg);skewX()从当前位置向X斜拉10度
transform: skewY(10deg);skewY()从当前位置向Y斜拉10度

5. 其他

transform: rotate() scale() translate();旋转、缩放、移动、倾斜、透视点根据组合需要选择使用
transform-origin: left top;改变中心点
perspective: 500;透视点，拥有3D效果（等于transform: perspective() + transform-style） 其他网站解释
transform: perspective();透视点（需要配合transform-style）
transform-style: preserve-3dz轴启用
perspective-origin: 50% 50%;改变 3D 元素的底部位置
transform: matrix(0, 0, 0, 0, 0, 0);六个参数的矩阵 其他网站解释
transform: matrix3d();九个参数的矩阵

Transition

transition-property应用过渡效果的 CSS 属性的名称
transition-property: none;没有任何css需要应用过渡效果
transition-property: all;所有的css都应用过渡效果
transition-property: property;指定的css应用过渡效果，多个css要用逗号或空格隔开
transition-duration: time;过渡效果花费的时间（以秒或毫秒计）
transition-timing-function规定过渡效果的速度曲线
transition-timing-function: linear;匀速
transition-timing-function: ease;快慢，快结束时变慢
transition-timing-function: ease-in;慢快
transition-timing-function: ease-out;快慢，到中间变慢
transition-timing-function: ease-in-out;慢快慢
: cubic-bezier(n,n,n,n);cubic-bezier(0.42, 0, 0.58, 1) 和ease-in-out一样 cubic-bezier(0, 0, 1, 1) 和linear一样
transition-delay: time;过渡效果何时开始（以秒或毫秒计）transition-delay: 1s, 2s;1s对应第一个需要延迟的css，2s对应第二个需要延迟的css
transitiontransition: property duration timing-function delay;简写属性，用于一次设置四个过渡属性。
transition: property duration;最少要写这两个，最简单的写法transition: all 2s;
分别使用transition: width 2s ease 2s, height 1s ease-in 1s;
